Welcome to p5.Polar (*´▽`*)

This is a game for learning how to draw patterns with p5.Polar library. The right hand side are example images. Follow the example images and type your code in the box below. You will learn how to draw different patterns through each levels.

No matter you have p5.js coding experience or not, p5.Polar library allows you to draw patterns from simple to complex easier with polar coordinate concept.

Code, Compile, Run
Type your answer in the box below in each levels. And run the code with Shift + Enter keys.
