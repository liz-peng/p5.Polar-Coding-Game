Welcome to p5.Polar (*´▽`*)

This is a game for learning how to draw patterns with p5.Polar library. The right hand side is example image that shows how you can draw a pattern from a single shape. The game requires you to follow the example images, type your code in the box below, and gradually draw the whole pattern with p5.Polar

Task
Draw a triangle by calling polarTriangle function

Hint
polarLine(angle, radius, distance)
polarTriangle(angle, radius, distance)

* Type your answer in the box below and execute the code with Shift + Enter keys
